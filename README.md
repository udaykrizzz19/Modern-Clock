## üéØ About The Project

This project is a comprehensive and interactive web-based clock application that perfectly replicates the sleek UI and robust functionality of modern mobile clock apps (Samsung Clock, iOS Clock). Built from scratch with enterprise-grade architecture, smooth animations, and seamless user experience that rivals native applications.

The application features a modular single-page architecture with four core modules: **Alarms**, **Timer**, **Stopwatch**, and **World Clock**, all unified through centralized state management and persistent storage.

### üèÜ Project Highlights

- **Production-Ready**: Fully tested, optimized, and deployed on Vercel
- **Native App Feel**: Locked viewport, smooth animations, and intuitive gestures
- **Enterprise Architecture**: Clean code, TypeScript, and scalable component structure
- **Cross-Platform**: Works seamlessly across desktop, tablet, and mobile devices
- **Accessibility First**: WCAG compliant with full keyboard navigation support

## üöÄ Getting Started

### Prerequisites

Ensure you have the following installed on your system:

- **Node.js** (v18.0.0 or later) - [Download here](https://nodejs.org/)
- **pnpm** (v8.0.0 or later) - [Installation guide](https://pnpm.io/installation)

### Quick Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/udaykrizzz19/Modern-Clock.git
   cd Modern-Clock
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Start development server**
   ```bash
   pnpm run dev
   ```

4. **Open your browser**
   
   Navigate to [http://localhost:5173](http://localhost:5173) to see the application.

### Available Scripts


## üèóÔ∏è Architecture & Design Decisions

### State Management Philosophy
- **Context API over Redux**: Lightweight, native React solution for global state
- **Custom Hooks**: Encapsulated business logic for timers, alarms, and stopwatch
- **Persistent Storage**: Automatic localStorage integration with custom hooks

### Accessibility & UX
- **Keyboard Navigation**: Full keyboard support for all interactions
- **Screen Reader Support**: Semantic HTML with proper ARIA labels
- **Motion Preferences**: Respects user's reduced motion preferences
- **Touch Optimized**: Large touch targets and gesture-friendly interactions

### Code Quality
- **TypeScript Strict Mode**: Comprehensive type safety throughout the application
- **ESLint + Prettier**: Consistent code formatting and style enforcement
- **Component Composition**: Reusable, composable components following React best practices
- **Error Boundaries**: Graceful error handling and recovery

### Precision Timer Logic

```typescript
// High-precision timer resistant to browser throttling
const useTimer = (duration: number) => {
  const [remainingTime, setRemainingTime] = useState(duration);
  const startTimeRef = useRef<number>(0);
  
  useEffect(() => {
    if (isRunning) {
      startTimeRef.current = Date.now();
      const intervalId = setInterval(() => {
        const elapsed = Date.now() - startTimeRef.current;
        const remaining = Math.max(0, duration - elapsed);
        setRemainingTime(remaining);
        
        if (remaining <= 0) {
          onComplete();
        }
      }, 50);
      
      return () => clearInterval(intervalId);
    }
  }, [isRunning, duration]);
};
```
